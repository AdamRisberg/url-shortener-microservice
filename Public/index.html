URL Shortener Microservice

To use this service, add /new/ followed by the URL you want to shorten to this site's address.

Create shortend URL example:

Request: www.example.com/new/http://www.google.com
Response: { "original_url": "http://www.google.com", "short_url": "www.example.com/1837" }

Shortened URL usage example:

www.example.com/1837
